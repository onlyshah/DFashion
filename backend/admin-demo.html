How to Start the Backend Server

Step 1: Install Node.js (if not already done)

  1. Go to nodejs.org
  2. Download the LTS version
  3. Install with default settings
  4. Restart your command prompt/PowerShell

Step 2: Start the Backend Server

  1. Open Command Prompt or PowerShell
  2. Navigate to the backend directory:
cd E:\DFashion\backend
  1. Run the server:
node test-server.js

Step 3: Access Admin Dashboard

  1. Open your browser
  2. Go to: http://localhost:5000/admin-login.html
  3. Login with any demo admin account
  4. Explore role-based dashboards!

Alternative: Continue Without Backend

Your Angular app works perfectly without the backend:

  • Angular App: http://localhost:4200
  • HTML Demo: Open demo-app/demo.html in browser
  • Full UI functionality with mock data
